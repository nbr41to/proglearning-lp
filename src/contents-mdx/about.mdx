## ¥1320/月から始めるプログラミング学習コミュニティ progLab

はじめまして！progLab は私がプログラミング学習のアウトプットのために始めた YouTube チャンネル[**progLearning**](https://www.youtube.com/channel/UCPcjWvYIfvqGPP4x30kEkMA)から派生してます.学ぶことが楽しくて、その楽しさを伝えたい！一緒に学びたい！という気持ちから発足に至りました！また、一緒に学習することで、挫折しにくく、なんだかんだ続けていられる環境を構成できたらいいなと思っています！

私は、もともと学校の先生をしていましたが、今はフリーランスエンジニアとして活動しております。1 年間で学ぶことが楽しいということと、凡人でもエンジニアになれることを証明しました。まだまだ未熟な凡人ですが、このことを多くの人に伝え、**一緒に成長する仲間の集まるコミュニティ**を作りたいと願い、このコミュニティを発足しました！

[応募はこちらから](/contact)

## 《Concept》

- コミュニケーションを提供することで、みなさんの成長をサポート
- 心理学や脳科学に基づく正しい学習方法や実践的なプログラミングの知識を提供
- 現役エンジニアが複数名参加

### こんな方は大歓迎 🙆‍♂️

## 《活動内容》

- 今週取り組んだことや学んだこと、来週取り組んで共有することを、毎週グループ通話で共有
- JavaScript（React）を用いた共同 Web アプリ開発
- around という通話アプリで、自分の学習記録を Slack 内で共有
- 時間が合えば気軽に通話しながら学習
- Slack 内では質問し放題
- 読書アウトプット会（7 月中に開始予定）

## 《学習内容》

Git/HTML/CSS/Sass/JavaScript/React.js/Node.js

モダンな技術である JavaScript のライブラリである React を軸とし、様々な開発の経験を積むことができ、フロントエンドエンジニアとして案件獲得ができるレベルまでサポートする体制が整っております。

学習内容は基本自由で、R 言語を学ぶ学生さんや WordPress/PHP で Web 制作をしている方もおりますので、上記の言語以外を学習して共有していただいても問題ありません！
