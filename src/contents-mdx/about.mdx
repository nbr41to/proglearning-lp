## 《学習内容》

学習内容は基本自由で、R 言語を学ぶ学生さんや WordPress/PHP で Web 制作をしている方もおります。
学んだ知識を逆に教えていただけると嬉しいです！

### サポート可能な言語/ツール

Git/HTML/CSS/Sass/JavaScript/React.js/TypeScript/Node.js

Next.js/styled-components/Recoil/ESLint/Prettier/Storybook/GraphQL/apollo-client/VSCode/Notion/...etc

主に、モダンな技術である JavaScript のライブラリである React を軸とし、
様々な開発の経験を積むことができ、
フロントエンドエンジニアとして案件獲得ができるレベルまでサポートする体制が整っております。
